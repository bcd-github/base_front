## 前端运行

```bash
# 克隆项目
git clone https://github.com/bcd-github/base_front.git

# 进入项目目录
cd base_front

# 安装依赖
npm install

# 启动服务
npm run dev

# 使用prettier格式化代码
npx prettier --write .

# 构建测试环境 npm run build:stage
# 构建生产环境 npm run build:prod
# 前端访问地址 http://localhost:80
```
